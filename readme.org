
* SymMat Implementation
See https://github.com/StewMH/GSoC2018/blob/master/evaluation_test.pdf for details

** Requirements
- =CMake=
- =C++11 Compiler=
- =Eigen3=

** Compilation 
To compile:
#+BEGIN_SRC sh
cd build
cmake ..
make
#+END_SRC

And then to run:
#+BEGIN_SRC sh
./symmat
#+END_SRC

Which should return:

=Test Passed=

Meaning the code at least works.
